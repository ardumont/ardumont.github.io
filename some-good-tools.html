Howto apt-get: default tool to install via command line

To search a package

apt-cache search name-of-app-needed

You can install everything with one command

sudo apt-get install list-of-packages

To know the list of packages already installed

dpkg -l | less

If you know what you search for, use grep to filter on the term researched 

dpkg -l | grep package-searched

To update all packages present on your machine

sudo apt-get update && sudo apt-get upgrade

Howto aptitude: little simpler installer command over apt-get

To search a package

aptitude search name-of-app-needed

You can install everything with one command

sudo aptitude install list-of-packages

To know the list of packages already installed

dpkg -l | less

If you know what you search for, use grep to filter on the term researched 

dpkg -l | grep package-searched

To update all packages present on your machine

sudo aptitude update && sudo aptitude safe-upgrade

Tools

Aptitude

sudo apt-get install aptitude

After that, you can almost use apt-get and aptitude the same way (minor some differences).

java sun

Know the jdk installed (openjdk by default on ubuntu, nothing on debian) 

sudo update-alternatives --list java

Install jdk sun 

sudo aptitude install sun-java6-jdk

or 

sudo apt-get install sun-java6-jdk

Choose the sun jdk: 

sudo update-alternatives --config java

Linux generic PAE

If you got over 4gb ram, install the PAE (Physical Address Extension). This permis to use from 4 a 64g de ram 

sudo apt-get install linux-generic-pae

As this is a kernel, you must reboot if you want to benefit from it.

Emacs: powerful text editor and much more

sudo apt-get install emacs

Git

sudo apt-get install git

Maven

sudo apt-get install maven

Squirrel-sql: client sql (mysql, postgresql)

For mysql:

Download the jar and launch it. http://www.squirrelsql.org/

Download the mysql connector http://dev.mysql.com/downloads/connector/j/5.0.html

Once installed, launch squirrel and add the mysql-connector jar into the driver panel.

Mysql-server

sudo apt-get install mysql-server
