Graphite

Definition

It's an open-source (Apache 2.0 license) enterprise-scale monitoring tool that runs well on cheap hardware which:

  • store numeric time-series
  • visualize graphs
  • It does not collect data.
  • Auto-monitoring

Mode

  • Push
  • File storage
  • URL API: graphs, json, csv
  • Functions -> MATHS!

Composed of

  • shisper
  • carbon
  • graphite-web

Flow

collectors -> metric -> | carbon -> whisper | -> write -> fs (cache) fs <- | whisper -> graphite-web |

carbon -> cache <- graphite-web

Carbon

Communication Protocols

TCP/UDP (preferred way)

  • the "plaintext protocol" 
echo "local.my.metric 123.45 `date "%s`" | nc location 2003
  • the "pickle protocol" 
[(path, (timestamp, value)), ...]
  • Advanced Message Queue Protocol

Whisper

Fixed-size database fast reliable storage

  • 1 metric = 1 file
  • structure

Samples of setup file: 

[carbon]
pattern = ^load\.
retentions = 10s:1d,1m:7d,4m30d

-> 346k 

[carbon]
pattern = ^load\.
retentions = 10s:15d,30s:180d,1m3y

-> 26 Mb 1 point every 10s for 15 days 1 point every 30s for 180 days …
