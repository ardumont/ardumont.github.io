Symptom

I couldn't install anything or upgrade anything on my linux box: 

sudo apt-get update && sudo apt-get upgrade
...
Reading database ... 60%dpkg: unrecoverable fatal error, aborting:
 files list file for package 'banshee' is missing final newline

Solution

Delete some messed up files for unknown reasons. #+BEGINSEC sh sudo rm /var/lib/dpkg/info/banshee.list sudo rm /var/lib/dpkg/info/banshee-extension-soundmenu.list #+BEGINSEC sh #+ENDSRC

Then force the reinstall of the package #+BEGINSEC sh sudo apt-get install banshee –reinstall #+ENDSRC

That's all folks
