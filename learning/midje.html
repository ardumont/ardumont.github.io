DONE An Emacs minor mode for Midje

^C k kill all rewards comments from midje ^C ^k Send everything down to the REPL (compile) ^C ,

  • Send the execution to the REPL (gets the result back just in front of the fact)

^C ^u

  • send the method into the (unfinished) part in your file
  • inside the (unfinished part), create the definition of the method not existing

^C-f Focus on the current fact ^C-h hides all facts ^C-s shows all facts
