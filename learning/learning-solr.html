Deleting Data

delete by id example java -Ddata=args -Dcommit=no -jar post.jar "<delete><id>SP2514N</id></delete>"

search the id that still exists before the commit id:SP2514N

commit java -jar post.jar

delete by query and commit java -Ddata=args -jar post.jar "<delete><query>name:DDR</query></delete>"
